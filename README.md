# briefai-news-digest
AI-powered news digest web app — fetch real headlines by topic and get instant AI summaries.
Features

1)Filter news by topic — Technology, World, Science, Business, Sports, Health
2)AI-generated 3-line summaries for each article
3)Dark / light mode toggle
4)Fully responsive — mobile and desktop
5)Save / bookmark articles


Tech Stack

HTML · CSS · Vanilla JS · NewsAPI · OpenAI API · GitHub Pages
